Autonomous Decision Triggers & Automation (15 Total)

Predictive Triggers → Slack Bot → Automated Workflows

IF CPA ↑ 25% WoW → THEN Pause/shift spend, investigate, launch diagnostic A/B test
IF Landing page conversion ↓ X% → THEN Launch A/B test, flag to marketing ops, budget hold
IF ROAS below threshold → THEN Reallocate to best channel, alert budget optimizer
IF Lead score high + intent spike → THEN Fast-track routing (VIP SLA), assign AE directly
IF Deal risk < 40 + discount requested → THEN Manager approval required, margin impact calculated
IF Pipeline coverage ratio < 3x → THEN Hiring flag raised, capacity model updated, CEO notified
IF Implementation utilization > 85% → THEN Hiring review triggered, timeline risk assessment sent
IF Contribution margin by segment < target → THEN Pricing review initiated, margin impact dashboard alert
IF Magic number drops below threshold → THEN Efficiency review triggered, cost reduction analysis
IF Delivery project margin negative → THEN Change order workflow, scope control intervention
IF Offline conversion rate drops → THEN Tracking/pixel audit, attribution verification
IF Usage drops 30% WoW → THEN CS alert + Slack, assign risk owner, renewal playbook
IF Renewal probability < 60% → THEN Risk workflow triggered, CS + AE assigned
IF P1 Implementation escalation → THEN Exec alert, auto-create CRM case, assign SLA timer
IF NRR drops below target → THEN Cohort analysis run, expansion opportunities identified, board alert

Computational Models: dbt Transforms → Intelligence Engine

Model Input Data Computation Logic Output Metrics
Revenue Model fct_opportunity, fct_invoice, fct_payment SUM(closed_won) by cohort/month/segment. Close date calc. Revenue recognition rules. ARR, MRR, ACV, TCV, Bookings, Billings, Collections %
Capacity Model dim_user, fct_opportunity, utilization logs Pipeline per AE/SDR. Projects per PM. Revenue per headcount. Utilization % (actual / available). Ramp by tenure. Coverage ratio, Revenue/AE, Utilization %, Hiring flag
Health Model fct_usage (daily agg), fct_ticket (TTR, severity), engagement flags Usage trend (WoW, MoM). Ticket + severity trend. Engagement score. Weighted health (0-100). Health score, Health trend, Engagement score, Risk category
Retention Model Health score, renewal history, price increases, ticket escalations Churn probability via logistic regression (features: health, usage drop %, ticket spike, price change). NRR = (Starting ARR + Expansion - Churn) / Starting ARR. Churn risk %, NRR %, GRR %, Expansion %, Cohort analysis
Attribution Model fct_lead (source, date), fct_opportunity (won, amount), ad spend, closed won + SQL (offline) First-touch: 100% to first source. Multi-touch: linear or time-decay. Offline: closed_won uploaded to ad platform. CPL, CPA, ROAS, Pipeline influenced ($), Attribution by channel
Unit Economics Model Revenue (by cohort), COGS, delivery hours (Rocketlane), support load, fully-loaded CAC (ads + SDR + AE comp) LTV = ARPU × gross_margin / monthly_churn_rate. CAC = (ad spend + sales cost + SDR salary allocation) / customers_acquired. Contribution margin = revenue - COGS - delivery - support. Magic number = quarterly ARR growth / sales & marketing spend. LTV, CAC, CAC payback months, Contribution margin %, Gross margin %, Magic number, Payback by segment
Marketing Experiment Model A/B test metadata (test date, variant, traffic split, conversions, spend), historical baseline Lift % = (variant conversion - control conversion) / control conversion. Confidence via binomial test. Segment lift (by audience, cohort, device, etc). Lift %, Confidence, Winning variant, Lift by segment, Rollout recommendation
Budget Optimization Model Ad spend by channel, fct_lead by channel, ROI/ROAS by channel, marginal CAC curve Estimate marginal CAC per channel from spend vs lead volume. Identify channel saturation (marginal CAC inflection). Allocate incremental budget to lowest marginal CAC (unconstrained optimization). Recommended spend allocation, Marginal CAC by channel, Channel saturation flags, Expected ROAS from reallocation
Scenario Simulator Model All models above + user inputs (headcount scenarios, discount policy, spend shifts, conversion assumptions) For each scenario: recalculate pipeline (capacity), revenue (pricing), delivery cost (utilization), contribution margin. Project ARR, NRR, cash impact, hiring needs. Forecast ARR, NRR impact, Capacity risk, Margin impact, Hiring requirements, Cash runway impact
Data Governance Model dbt manifest, table/column metadata, metric definitions, RBAC matrix Freshness check (last_modified timestamp vs SLA). Schema validation (expected columns). Metric lineage (source → transform → output). RBAC enforcement (finance metrics require CFO access). Data quality score %, Freshness status, Lineage clarity, RBAC compliance, Audit trail

Intelligence Engine: Predictive Algorithms (8 Core)

Algorithm Input Signals Logic Output Action
Anomaly Detection Usage (daily), Reply rate, SLA breaches, CPA, Discount rate Z-score or isolation forest on rolling 30-day baseline. Flag if 2σ deviation. Alert to owner (CS for usage drop, Marketing for CPA spike, Finance for discount spike)
Forecast Deviation Weekly forecast vs actual pipeline stage change Variance % = (Actual - Forecast) / Forecast. If > 10%, flag. VP Sales weekly review; deal-by-deal risk analysis
Deal Risk Scoring Rattle signals, Outreach activity, Days in stage, Discount applied, Deal size Logistic regression on historical won/lost. Score 0-100. Red flag if < 40. Heightened engagement; manager approval for discount if risk < 40
Cohort Retention Analysis NRR model output, by acquisition cohort (month, segment, region) Cohort survival curve. Identify declining cohorts. Compare to baseline. Board alert if cohort NRR trending down; trigger retention task force
CAC Payback Projection Monthly cohort revenue, cash received timeline, historical payback curves Extrapolate payback months forward. Flag if trending beyond target. Finance alert; trigger cost reduction or pricing adjustment review
Margin by Segment & Unit Econ Bookings, COGS, delivery cost, support cost, fully-loaded CAC by segment Gross margin % and contribution margin %. Identify low-margin segments. Magic number trend. Pricing review for low-margin segments; focus on high-margin expansion; efficiency review
Experiment Lift Detection (NEW) A/B test results (conversions, lift %, confidence), historical baseline Binomial test for statistical significance. Segment lift analysis. Rollout readiness score. Confidence-based rollout recommendation; budget allocation to winning variant
Scenario Impact Modeling (NEW) All models + user scenario inputs (headcount, discount policy, spend, conversions) Recalculate pipeline, revenue, delivery cost, NRR, cash. Show delta vs baseline. CEO can query "what if we hire 2 AEs?" or "cap discounts at 15%?" and see impact
